Pillar 3: Automated Response Preparation Engine

Action 3.1: Build Make.com Automation: Core Plan (~$11/mo). Scenario: Feedly RSS -> Filter -> Parse -> Search KB -> Get Snippet/Link -> OpenAI API -> Send Draft to Review Queue. Refine OpenAI prompt over time.

Conceptual Flow Diagram:

  1. Feedly detects Reddit mention -> RSS.
  2. Make triggers.
  3. Make filters.
  4. Make parses text.
  5. Make searches KB.
  6. Make retrieves Snippet/Link.
  7. Make calls OpenAI.
  8. OpenAI generates Draft.
  9. Make sends to Review Queue (Slack).
  10. (Manual) CSS Team reviews.
  11. (Manual) CSS Team edits/posts.
